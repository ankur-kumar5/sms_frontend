# Salary Management System Frontend Service
This is the frontend service for the Salary Management System. It provides a user interface for employees to manage their information, and perform related tasks.
## Features
- Employee Information Management: Employees can view and update their personal information, such as full name, country, and job title.

## Technologies Used
- React: A JavaScript library for building user interfaces.
- Axios: A promise-based HTTP client for making API requests.
- React Router: A library for routing in React applications.

## Installation
1. Clone the repository:
   ```bash
    git clone <repository_url>
    ```
2. Navigate to the project directory:
    ```bash
    cd sms_frontend
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev     
    ```
5. Open your browser and navigate to `http://localhost:5173` to access the application.
